JQuery slideToggle Navigation
=============================

Doing some practice with JQuery and the slideToggle function.

```js
$(".menu_btn").click(function(){
  $("#menu").slideToggle();
  $(this).toggleClass("opened");
});
```

## Built With

* JQuery (1.10.2)
* Bootstrap (3.1.1)
